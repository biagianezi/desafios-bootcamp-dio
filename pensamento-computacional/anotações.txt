	Lógica de programação:
Lógica é a organização e planejamento das instruções assertivas em um algoritmo a fim de resolver um  problema.
Técnica linear – execução sequenciada (acordar, descer as escadas, preparar o café, tomar o cafezinho) 1 única dimensão
Tecnica estruturada – podemos ter conduções, podemos ter mais de uma opção, ao invés de ter a opção de só fazer café, pode ser suco. 
Tecnica modular – dados de entrada – processo de transformação – dados de saída 
Definimos partes independendtes que são controladas por um conjunto de regras onde cada módulo tem suas regras específicas. 

Variáveis = tipo de estrutura mutável, insconstante, pode receber mais de um valor e ser modificada. Ela não sabe  valor do que vai receber, mas sabe o tipo do dado.
Tipos de dados = númerio (inteiros e reais )
Caracteres =  ex: str
Lógico = booleano
O nome da variável segue algumas regras :
1 ou mais caracteres 
Primeira letra tem que ser LETRA
Sem espaços em branco
Vedado utilizar palavras reservadas de algumas linguagens
Variavel vai ser se ação ou de controle
Constante = é inalterável. = variável que não muda
Instruções primitivas = dentro de cada linguagem de programação possui uma sintaxe particular
Dados são importados de algum lugar, são processados e depois imprmidos de alguma forma.
Estruturas condicionais e operadores = Dado uma condição que foi satisfeita a operação é executada. 
Simples (se)
Composta (senão)
Encadeada (se não se)
Estruturas de repetição = condição de parada 
Exemplos: enquanto ... faça
Repita... ate
Para... de ... até... faça – for
Vetores e matrizes =  vetor é uma sequencia de tipos de dados = uma variável que possui um tamanho pre fixado
Matriz é uma tabela localizada com linhas e colunas
